Stage Calculator

A simple calculator to calculate all you need for a stage. However its only for vanilla KSP and does not yet support any custom parts.

Once the KSP Main directory is added to the configuration, its will be possible to use custom parts as well.

The Calculator will return the following Values:

  • The full and empty weights of the stage
  • The Specific Impulse of the stage
  • The thrust of the stages engines
  • The presumed Δv of the stage
